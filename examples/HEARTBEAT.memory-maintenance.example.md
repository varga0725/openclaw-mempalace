# HEARTBEAT memory maintenance example

Use this kind of heartbeat guidance when you want lightweight memory review without moving everything into cron.

1. Check whether there were important new decisions, rules, or workflows today.
2. Check whether anything should be written into daily memory files or curated memory.
3. Check whether MemPalace-related work created follow-up items, bugs, or docs debt.
4. If nothing new happened, reply exactly `HEARTBEAT_OK`.

Do not invent memories. Do not create fake summaries. Only surface real maintenance work.
